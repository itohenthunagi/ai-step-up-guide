⚡ 便利機能・拡張

各サービスの便利機能を分かりやすい比喩で解説。もっと効果的にAIを活用しましょう！

🤖 ChatGPT Plus

🛍️ GPTs & ストア

他ユーザーが作ったAIアシスタント（GPT）を自由に呼び出して使える機能です。

💭 比喩: いわば「AIのアプリストア」のようなもの。料理レシピ専門AI、英語学習AI、プログラミング支援AIなど、目的別に特化したAIが揃っています。

📊 Advanced Data Analysis

CSVやExcelファイルのデータを簡単に視覚化できる機能です。

💭 比喩: 専属データアナリストのような存在。売上データをアップロードするだけで、グラフ作成から傾向分析まで自動で行ってくれます。

🎙️ 音声入力 & 画面共有

スマホアプリで友人に話す感覚で音声入力が可能。画面を共有しながら会話もできます。

💭 比喩: リモート相談のような便利さ。「この画面のここが分からない」と画面を見せながら質問できるので、非常に直感的です。

🔍 Perplexity Pro

🎯 Focus モード

検索対象を学術論文（Academic）やニュース（News）などに絞れる機能です。

💭 比喩: 図書館の専門コーナーのような機能。医学書コーナーや新聞・雑誌コーナーのように、信頼性の高い情報源だけを選んで調査できます。

🌐 Spaces 共有

検索結果や調査内容をオンラインで簡単に共有できる機能です。

💭 比喩: オンライン掲示板のような使い勝手。チームメンバーと調査結果を共有したり、議論を続けたりできます。

📚 NotebookLM

📋 Auto Source Guide

文書をアップロードすると重要な要点を自動抽出・要約してくれる機能です。

💭 比喩: 優秀な秘書が文書整理してくれる感覚。長い報告書や資料を渡すと、重要なポイントだけを整理してくれます。

🎧 音声要約再生

長い文書の要約をポッドキャスト形式で聴ける機能です。

💭 比喩: 通勤時のラジオ番組のような機能。複雑な資料も音声で要約してくれるので、移動中でも効率的に情報収集できます。

💡 便利機能の活用シーン

📊 データ分析

ChatGPT PlusのAdvanced Data Analysisで売上データや顧客データを簡単に可視化・分析できます。

🔍 専門調査

Perplexity ProのFocusモードで学術論文や最新ニュースに限定した信頼性の高い情報収集が可能です。

📚 資料整理

NotebookLMで大量の資料を一元管理し、音声要約で効率的に内容を把握できます。
